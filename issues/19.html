Quick question as Tianna and I are doing markup today: should we tag all unicode symbols with the figure tag? Is that what you use to pull the figure metadata, or was that done manually?


tcatapano commented:

@TillmannTaape: You should not tag symbols as <figure>. Symbols for which unicode characters are being used have been tagged as <figure> but these will be automatically converted to <mark>. We really dont have any specs as to how this tag is going to used right now, so its a very low priority.
