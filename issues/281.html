#281: Support and rendering for "external" resources/references

opened by njr2128

Definition of "external" resources/references

Anything that is outside of the current M&K GD annotation structure and elements (text, bibliography, abstract, illustrations, captions)

Examples of "external" resources/references

  • Appendices
  • Field notes
  • Hyperlinks within annotation text (or other elements) to externally-hosted resources (e.g. link to museum page, or to another project)
  • Hyperlinks to M&K resources outside of annotations (e.g. Student Files materials/research doc, or BnF folder resources)

Problem to be considered

  • How will some of these be supported? e.g.

    if hyperlink to externally-hosted page, rely on that hosting institution if appendix, appendices need to be integrated into annotation structure if hyperlink to M&K resource, ensure it can be supported long-term (like solution for field notes)

  • How will some of these be rendered? e.g.

    appendices will appear before/after footnotes and bibliography?
