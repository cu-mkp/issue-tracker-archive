Instructions and larger issues that have been raised (see second sheet "Instr. & Ass." of DCE_comment-tracking): ## Version of text (tc, tcn, tl) For the most part, pull comments across all three versions. However, this may not always make sense - use discretion and come to a decision after discussion if necessary. PHS will review each case carefully and determine final status. ## Categories Assign each comment one of the following categories: category | description -- | -- editorial | explaining editorial decision or interpretation of the text and its semantics, e.g. uncorrected errors on AP's part historical | historical background (like author page) - change category title to also include larger context informational, e.g. latin name of plants structural | material page or structure of ms conjectural | editorial conjecture translation | we translated because... handshift | mostly from MHS ## References - when annotation, include ID in reference column. Leave out titles -when referencing another folio, use folio id syntax (003r, 012v, 134r) and enter ID in reference column - when referencing another entry, use folio id in the comment, but link entry id in reference column and use entry id syntax (p003r_1, p012v_2, p134r_3) - external references, such as citations, books, etc - TBD - related to glossary references, see #985 ## Language and formatting - be as accurate and clear as possible - if the comment is a full sentence: first word should be capitalized, end comment in period (if only a couple of words, no period is necessary) - italics: indicate with `<i>` and `</i> around the words to be italicized - use quotation marks (" - double, not single ticks) when needed - "i.e.," - always with a comma as shown here - use "author-practitioner" to refer to writer of Fr. 640 - all foreign languages that are translated in a comment should be of the format: [Language]: [translation] - Occitan terms should be treated as foreign language. Poitevin should not (words are marked up and dealt with in the glossary like other non-translated French terms) - to indicate distance, use km (not miles)
